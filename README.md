# 房地产股票 AI 评级 Agent

基于**量化技术分析 + 腾讯混元2.0大模型**的中国房地产股票智能评级系统，覆盖 A股、港股、美股三大市场共 55 只房地产相关股票。

## 评级模型

### 综合评分 = 量化技术评分 × 50% + AI大模型评分 × 50%

**量化技术评分**（5个维度加权）：

| 维度 | 权重 | 指标 |
|------|------|------|
| 趋势评分 | 25% | 四级均线排列(MA5/10/20/60)、价格偏离度、MA20斜率、均线黏合度、ADX趋势强度 |
| 动量评分 | 20% | RSI双周期(6+14)、MACD信号+柱状体变化、KDJ随机指标、Williams %R、多周期涨跌幅 |
| 波动率评分 | 15% | 年化波动率、布林带宽度+价格位置、ATR(14)相对波动、波动率收敛/发散趋势 |
| 成交量评分 | 20% | 多级量比(5/10/20日)、OBV能量潮、VWAP偏离度、量价配合度、成交量趋势 |
| 价值评分 | 20% | 区间位置连续评分、筹码集中度、多级支撑压力(10/20/60日)、价格动态区间 |

**AI大模型评分**（腾讯混元2.0 Think）：
- 公司基本面（权重40%）：经营质量、财务健康(三道红线)、土储质量、销售回款、管理层能力
- 行业政策与周期（权重30%）：房地产调控政策、行业周期阶段、融资环境、市场供需格局
- 技术面与资金面（权重20%）：价格趋势、机构资金动向、北向资金、筹码结构分析
- 风险评估（权重10%）：债务违约风险、项目交付风险、系统性行业风险

**评级映射**：

| 综合评分 | 评级 |
|----------|------|
| ≥ 80 | 强烈推荐 |
| 65 - 79 | 推荐 |
| 50 - 64 | 中性 |
| 35 - 49 | 谨慎 |
| < 35 | 回避 |

> 若 AI 不可用，自动降级为 100% 量化评分。

## 技术栈

**后端**：
- FastAPI + Uvicorn（异步 Web 框架）
- SQLAlchemy + aiosqlite（异步 ORM + SQLite）
- APScheduler（定时任务，每日 09:00 自动评级）
- AKShare（A股/港股/美股行情数据）
- httpx（异步调用腾讯混元 API）
- pandas / numpy（量化计算）

**前端**：
- React 18 + Vite（构建工具）
- Recharts（价格走势和评分趋势图表）

**部署**：
- Docker + Docker Compose
- Nginx 反向代理

## 快速开始

### 1. 环境准备

```bash
# 克隆项目
git clone <repo-url>
cd "real estate stock agent"
```

### 2. 配置腾讯混元 API

```bash
cd backend
cp .env.example .env
```

编辑 `backend/.env`，填入你的密钥：

```env
HUNYUAN_SECRET_ID=你的SecretId
HUNYUAN_SECRET_KEY=你的SecretKey
HUNYUAN_MODEL=hunyuan-2.0-thinking-20251109
```

### 3. 本地开发运行

**启动后端**：

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**启动前端**：

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

### 4. Docker 部署

```bash
docker-compose up -d
```

访问 http://localhost

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/dashboard` | 仪表盘统计数据 |
| GET | `/api/stocks` | 股票列表 |
| GET | `/api/ratings/latest` | 最新评级（支持筛选和排序） |
| GET | `/api/ratings/dates` | 可用评级日期列表 |
| GET | `/api/ratings/date/{date}` | 按日期查询评级 |
| GET | `/api/ratings/history/{code}` | 单只股票评级历史 |
| GET | `/api/prices/{code}` | 股票价格数据 |
| GET | `/api/rating-trend/{code}` | 评分趋势数据 |
