# 房地产股票 AI 评级 Agent

基于**量化技术分析 + 情绪因子 + 同花顺iFinD专业数据 + 腾讯云三大模型(DeepSeek V3.2 / GLM-5 / Kimi K2.5) + 实时新闻资讯**的中国房地产股票智能评级系统，覆盖 A股、港股、美股三大市场共 60 只房地产相关股票。支持**双评级模型**（量化AI选股 / 东吴地产选股）、**自选股票池**、多数据源自动降级、AI动态权重调整。

## 功能模块

### 1. 用户认证系统
- JWT Token 认证，支持多终端同时登录，token 有效期 72 小时
- 管理员后台创建用户，最大支持 100 个账户
- 角色权限：管理员（全部功能）/ 普通用户（查看功能）

### 2. AI 智能评级（双模型架构）

系统内置两套评级模型，前端一键切换：

**模型一：量化AI选股**
- **综合评分 = 量化技术 × 25% + 情绪因子 × 10% + iFinD基本面 × 15% + AI大模型 × 50%**
- 量化技术6维度：趋势/动量/波动率/成交量/价值/情绪
- 情绪因子：关键词情绪分析 + 双速动态衰减模型 + AI情绪反馈融合
- AI动态权重调整：三模型输出weight_hints，动态调整各量化维度权重（±30%封顶）
- 若基本面不可用则自动降级为量化30%+情绪12%+AI58%

**模型二：东吴地产选股**
- **综合评分 = iFinD基本面 × 70% + AI大模型 × 30%**
- 聚焦房地产行业基本面：宏观LPR利率、行业政策信号、三道红线、PB<1估值安全边际
- 基本面四维度：三道红线(30分)+估值安全边际(30分)+盈利质量(20分)+资金面(20分)
- AI聚焦：宏观政策(30%)+行业信号(25%)+公司财务(25%)+估值与风险(20%)

**共同特性**：
- **三模型融合**：DeepSeek V3.2 (40%) + GLM-5 (30%) + Kimi K2.5 (30%)，支持部分模型失败时动态权重归一化
- 每日 09:00 自动刷新，覆盖 60 只房地产股票
- **iFinD专业数据**：PE/PB/ROE/EPS/资产负债率/资金流/量比/委比/振幅/换手率/连涨跌天数/多周期涨跌幅
- **基本面5维评分**：核心估值(50分)+资金面(20分)+市场情绪(10分)+盈利能力(10分)+交易活跃度(10分)
- **港股数据增强**：iFinD不返回连涨天数和多周期涨跌幅时，自动由历史行情计算补充
- **换手率合理性校验**：自动过滤异常值（如港股返回的非百分比原始数据）
- **混元联网搜索**：开启 enable_search，AI 可实时搜索最新政策信息
- 评分精度精确到小数点后 2 位
- 支持按市场、评级筛选，支持历史日期回溯

### 3. 股票详情面板（iFinD数据驱动）
- **核心估值**：PE(TTM)、PB(MRQ)、总市值、ROE、EPS、资产负债率、基本面评分
- **多周期涨跌幅**：5日/10日/20日/60日/120日/年初至今，红绿色直观显示
- **资金流向**：主力净流入（含进度条可视化）、散户净流入、超大单净流入
- **市场微观数据**：量比、换手率、委比、振幅、连涨/跌天数
- **公告与财报**：近90天公告，财报/重点公告自动标记，支持PDF下载；点击个股时预加载公告数据
- **AI专业分析**：腾讯云三模型融合分析报告，含评分构成公式
- 价格走势图表、评分趋势、历史评级记录

### 4. 地产行业要闻
- **新闻+公告合并展示**：行业新闻 + iFinD公告统一时间线
- **默认展示5条，点击"查看更多"扩展至10条**
- **30天时效过滤**：自动排除旧闻，按时间倒序排列
- **官方权威数据源**：中国政府网、人民网、新华网 + 东方财富 + 同花顺iFinD
- **AI 智能筛选**：混元大模型评分，仅保留高相关度内容（15秒超时自动降级为关键词排序）
- **启动预热缓存**：服务启动时后台预加载新闻缓存，用户首次访问秒级响应
- 5分钟自动刷新

### 5. 自选股票池
- 用户可从 60 只房地产股票中自选最多 **10 只**股票加入个人关注池
- 支持搜索添加、备注编辑、一键删除
- **AI操作建议**：三模型融合生成个性化操作建议（买入/持有/减仓/观望），含具体理由
- 建议以卡片式展示，不同操作建议用颜色区分

### 6. 多数据源降级
- **四层数据架构**：iFinD专业数据 → akshare（东方财富）→ 腾讯财经 → 新浪财经
- **iFinD覆盖**：历史行情、实时行情（80+指标）、估值、财务、公告
- **智能数据源排序**：连续失败超过 3 次的数据源自动降低优先级

### 7. 市场点评
- 管理员发布每日行业点评和个股点评
- 支持分类筛选、创建、编辑、删除
- **支持分享到微信朋友圈**（生成公开链接，无需登录即可查看）

### 8. 研究报告
- 管理员上传机构研究报告（PDF/Word/Excel/PPT，最大50MB）
- **PDF在线预览**（RFC 5987中文文件名编码），上传成功动画反馈
- **支持分享到微信朋友圈**（含报告摘要、PDF内嵌预览和下载入口）

## 评级模型详解

系统内置两套独立评级模型，前端一键切换，数据库按 `model_type` 字段分别存储。每日 09:00 自动对 60 只股票执行双模型评级。

---

### 模型一：量化AI选股 (`quant_ai`)

**综合评分 = 量化技术 × 25% + 情绪因子 × 10% + iFinD基本面 × 15% + AI大模型 × 50%**

> **降级策略**：若基本面不可用，自动降级为量化30%+情绪12%+AI58%；若AI不可用，降级为量化50%+情绪10%+基本面40%；若AI和基本面均不可用，降级为量化88%+情绪12%。

#### 1. 量化技术评分（6个维度加权，满分100分）

**趋势评分（权重22%）**：
- **四级均线排列**（0\~30分）：判断 MA5/MA10/MA20/MA60 的多头/空头排列程度，每一级顺序正确+10分，逆序-10分
- **价格偏离度**（0\~20分）：当前价格相对 MA20 的偏离百分比，偏离-10%得0分，+10%得满分，线性映射
- **MA20斜率**（0\~15分）：取最近10日 MA20 的变化率，斜率-5%得0分，+5%得满分
- **均线黏合度**（0\~15分）：四条均线标准差/MA20，黏合度<2%得12分（突破潜力），>6%得2分
- **ADX趋势强度**（0\~20分）：ADX>25且+DI>-DI时按 ADX 值线性给高分（强上升趋势），ADX>25但-DI更大时给低分（强下降趋势），ADX<25给中性8分

**动量评分（权重18%）**：
- **RSI双周期**（0\~20分）：RSI(14)贡献0\~12分 + RSI(6)贡献0\~8分，从超卖(20)到超买(80)线性映射
- **MACD信号**（0\~25分）：DIF零轴位置(0/8分) + 金叉/死叉状态(2/7分) + 柱状体变化方向(1\~10分，红柱放大=加速上涨得满分)
- **KDJ随机指标**（0\~20分）：K>D且J>0时高分，K<D且J<0时低分；K/D同时>80超买区减3分，<20超卖区加2分
- **Williams %R**（0\~10分）：WR从-80(超卖)到-20(超买)线性映射
- **多周期涨跌幅**（0\~15分）：5日/10日/20日各贡献0\~5分，短中期涨幅越大得分越高

**波动率评分（权重12%）**：
- **年化波动率**（0\~25分）：20日年化波动率，15%以下满分，60%以上最低分（低波动=风险可控=高分）
- **布林带宽度+位置**（0\~25分）：带宽评分(0\~12分，窄带更稳定) + 价格位置评分(0\~13分，中轨0.3\~0.7区间最佳得13分)
- **ATR(14)相对波动**（0\~25分）：ATR/价格百分比，1%以下满分，6%以上最低分
- **波动率趋势**（0\~25分）：对比近10日与前20日波动率，收敛>5%得22分，扩大>5%得4分

**成交量评分（权重18%）**：
- **多级量比**（0\~20分）：5日/20日量比1.0\~1.5为温和放量得12分，>2.5异常放量得3分；10日/20日量比>1.0加8分
- **OBV能量潮**（0\~25分）：OBV>MA5>MA20（资金持续流入）得18分，OBV<MA5<MA20（持续流出）得3分；OBV斜率附加0\~5分
- **VWAP偏离度**（0\~15分）：价格高于VWAP 0\~3%为温和强势得15分，偏离>6%过度偏离得5分，低于VWAP>3%得3分
- **量价配合度**（0\~20分）：上涨日均量/下跌日均量>1.3得18分（上涨放量下跌缩量），<0.7得3分（不健康）
- **成交量趋势**（0\~10分）：10日/30日均量比线性映射

**价值评分（权重18%）**：
- **区间位置**（0\~25分）：价格在历史高低点间的位置，0.2\~0.5区间（低位有支撑）得25分，>0.85极高位得5分
- **筹码集中度**（0\~20分）：30日成交量加权价格标准差/VWAP，越集中得分越高
- **多级支撑压力**（0\~25分）：10日(3分)/20日(5分)/60日(7分)三级，距支撑近+距压力远得满分
- **价格动态区间**（0\~30分）：10日变异系数<2%得18分（窄幅盘整蓄力）；5日涨跌幅附加0\~10分

**情绪评分（权重12%）**：
- **关键词情绪分析**：30个正面关键词（放松限购、降息、利好、增持等）+ 25个负面关键词（爆雷、违约、暴跌、减持等），房地产行业专属
- **双速动态衰减模型**：新闻快衰减（半衰期3天）、公告慢衰减（半衰期7天），公式：`weight = 0.5^(days_ago / halflife)`
- **AI情绪反馈融合**：AI独立输出sentiment_score(0-100)，最终情绪 = AI×60% + 关键词×40%
- 返回0-100分：50为中性，>50偏多，<50偏空

#### 2. iFinD基本面评分（五维度，满分100分，动态归一化）

| 维度 | 满分 | 评分细则 |
|------|------|----------|
| 核心估值 | 50分 | **PE(TTM)** 15分: <8极低PE得15, 8\~15得12, 15\~30得8, 30\~50得4, >50得1, 亏损得2 |
| | | **PB(MRQ)** 14分: 0.5\~1.0破净得14(修复潜力最大), <0.5得11, 1.0\~1.5得10, >2.5得1 |
| | | **ROE** 11分: >15%得11, 8\~15%得9, 3\~8%得7, 0\~3%得3, <0%得1 |
| | | **资产负债率(三道红线)** 10分: <70%绿档得10, 70\~75%得7, 75\~80%得5, 80\~85%得2, >85%得1 |
| 资金面 | 20分 | **主力净流入** 10分: >5000万得10, 1000\~5000万得8, 0\~1000万得6, 流出得2\~4 |
| | | **量比** 6分: 0.8\~1.5温和放量得6, 1.5\~2.5得4, >2.5异常放量得2 |
| | | **委比** 4分: >30%强买盘得4, >10%得3, ±10%平衡得2, <-30%强卖盘得0 |
| 市场情绪 | 10分 | **连涨天数** 5分: ≥5天得5, 3\~4天得4, 1\~2天得3, 连跌得0\~1 |
| | | **振幅** 5分: <2%极稳得5, 2\~4%得4, >6%高波动得1 |
| 盈利能力 | 10分 | **EPS** 10分: >1.5元优秀得10, 1.0\~1.5得8, 0.5\~1.0得6, 0\~0.1得2, 亏损得0 |
| 交易活跃度 | 10分 | **换手率** 5分: 1\~5%适中得5, 5\~10%偏高得4, >10%投机得2, <0.5%极低得1 |
| | | **20日涨跌幅** 5分: 5\~20%温和上涨得5, 0\~5%小幅得4, >40%或<-15%极端得0 |

> 动态归一化：实际可用维度得分/实际可用维度满分×100，确保港股等缺少部分指标时评分合理。

#### 3. AI大模型评分（三模型融合 + 联网搜索）

系统采用 **DeepSeek V3.2 (40%) + GLM-5 (30%) + Kimi K2.5 (30%)** 三模型融合架构，均通过腾讯云 LKEAP OpenAI 兼容接口调用。当部分模型不可用时，剩余模型权重自动归一化。

模型接收完整的 **行情数据摘要 + iFinD财务指标 + 行业新闻 + 公司公告 + 量化评分** 作为上下文。

**输入数据组成**：
- 股票基本信息 + 最新价格、多周期涨跌幅、区间高低点、波动率、量比
- iFinD财务数据：PE/PB/ROE/EPS/负债率/总市值/换手率/资金流/涨跌幅
- 行业新闻摘要：由新闻爬虫获取的最新房地产政策、行业动态
- iFinD公司公告：近30天的公告标题+日期
- 量化评分参考值（6个维度评分，含情绪因子）

**AI评分维度**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 政策资讯影响 | 35% | 联网搜索最新政策，评估限购/限贷/利率/城中村改造等对个股的具体影响；重大利好可显著上调评分 |
| 公司基本面 | 30% | 经营质量、三道红线、土储城市布局、销售回款、管理层战略执行力 |
| 技术面与资金面 | 25% | 中长期均线方向、成交量变化反映的机构态度、北向资金、筹码结构 |
| 风险评估 | 10% | 债务违约、项目交付、宏观经济下行、估值泡沫、流动性风险 |

> AI输出格式：`{"ai_score": 0-100, "sentiment_score": 0-100, "weight_hints": {"trend": 0-1, ...}, "analysis": "250字以内专业分析"}`
>
> **AI动态权重调整**：AI通过weight_hints评估各量化维度的当前可靠度，系统据此动态调整量化内部权重（乘数0.7\~1.3，±30%封顶后重归一化）。三模型的weight_hints取加权平均以确保稳定性。

#### 4. 综合评分与降级

```
正常:    综合 = 量化×25% + 情绪×10% + 基本面×15% + AI×50%
无基本面: 综合 = 量化×30% + 情绪×12% + AI×58%
无AI:    综合 = 量化×50% + 情绪×10% + 基本面×40%
全降级:  综合 = 量化×88% + 情绪×12%
```

### 模型二：东吴地产选股 (`soochow`)

**综合评分 = iFinD基本面 × 70% + AI大模型 × 30%**

> **设计理念**：该模型聚焦房地产行业基本面，不使用量化技术维度。核心选股条件：宏观LPR利率下行、行业政策宽松信号、三道红线达标、PB<1估值安全边际。
>
> **降级策略**：若AI不可用，100%基本面；若基本面也不可用，返回空（不评级）。

#### 1. 东吴基本面评分（四维度，满分100分，动态归一化）

**三道红线 & 财务健康（30分）**：
- **资产负债率**（0\~15分）：<70%绿档得15分，70\~75%得11分，75\~80%得7分，80\~85%得3分，>85%严重超标得1分
- **ROE盈利能力**（0\~15分）：≥15%得15分，10\~15%得12分，5\~10%得9分，0\~5%得5分，亏损得1分

**估值安全边际（30分）**：
- **PB估值**（0\~20分）— 核心条件：PB 0.5\~0.8为最优区间得20分（破净+合理范围），<0.3深度破净得15分（可能有风险），0.8\~1.0得16分，>1.5得2分
- **PE估值**（0\~10分）：<8得10分，8\~15得8分，15\~25得5分，>40得1分，亏损得2分

**盈利质量（20分）**：
- **EPS每股收益**（0\~12分）：>2.0得12分，1.0\~2.0得10分，0.5\~1.0得7分，0.1\~0.5得4分，亏损得0分
- **20日涨跌幅趋势**（0\~8分）：5\~20%温和上涨得8分（趋势向好），0\~5%得6分，-5\~0%得3分

**资金面（20分）**：
- **主力净流入**（0\~12分）：>5000万得12分，1000\~5000万得10分，0\~1000万得7分，流出得1\~4分
- **换手率**（0\~8分）：1\~5%适中活跃得8分，5\~10%偏高得6分，<0.5%极低得2分

#### 2. 东吴AI评分（三模型融合 + 联网搜索）

专用 System Prompt 模拟地产研究团队视角，采用 DeepSeek V3.2 + GLM-5 + Kimi K2.5 三模型融合：

| 维度 | 权重 | 分析框架 |
|------|------|----------|
| 宏观环境 | 25% | 5年期LPR利率走势（下行利好）、社融规模环比变化、降准降息预期、房贷利率 |
| 行业政策 | 30% | 需求端（放松限购/降首付/降利率）、供给端（保交楼/白名单/城中村改造）、政策基调（"止跌企稳"/"政策宽松"）、重点城市落地情况 |
| 个股基本面 | 30% | 营收同比变化、存货/土地减值计提、三条红线达标（剔除预收后负债率<70%/净负债率<100%/现金短债比>1）、经营性现金流 |
| 估值安全边际 | 15% | PB<1为核心条件、PE是否历史低位、股息率吸引力 |

> AI输出格式：`{"score": 0-100, "analysis": "详细分析"}`

#### 3. 综合评分与降级

```
正常:    综合 = 基本面×70% + AI×30%
无AI:    综合 = 基本面×100%
无基本面: 综合 = AI×100%
全不可用: 不评级（返回空）
```

---

### 评级映射（两模型通用）

| 综合评分 | 评级 | 含义 |
|----------|------|------|
| ≥ 65 | 优选 | 多维度表现优异，具备较强投资价值 |
| 50 - 64 | 关注 | 整体表现良好，值得持续跟踪 |
| 35 - 49 | 中性 | 表现一般，建议观望 |
| < 35 | 谨慎 | 风险较高，建议规避 |

### 数据流转全景

```
定时任务(09:00) → 逐只股票处理:
  ├─ 1. 获取历史行情(120日) ← iFinD → akshare → 腾讯 → 新浪 (四层降级)
  ├─ 2. 保存价格数据到 stock_prices 表
  ├─ 3. 量化AI模型:
  │   ├─ 获取新闻/公告 → 计算关键词情绪因子(双速衰减)
  │   ├─ 计算6维量化评分(趋势/动量/波动率/成交量/价值/情绪)
  │   ├─ 获取iFinD基本面 → 计算5维基本面评分
  │   ├─ 拼装AI提示词 → 三模型并行调用 → 融合评分+情绪反馈+权重调整
  │   └─ 加权合成 → 写入 ratings 表 (model_type='quant_ai')
  ├─ 4. 东吴地产模型:
  │   ├─ 获取iFinD基本面 → 计算4维东吴基本面评分
  │   ├─ 拼装东吴AI提示词 → 三模型并行调用 → 融合评分
  │   └─ 加权合成 → 写入 ratings 表 (model_type='soochow')
  └─ 5. 等待2~4秒 → 下一只股票
```

## 技术栈

**后端**：
- FastAPI + Uvicorn（异步 Web 框架）
- SQLAlchemy + aiosqlite（异步 ORM + SQLite）
- APScheduler（定时任务，每日 09:00 自动评级）
- **同花顺iFinD HTTP API**（历史行情/实时行情/估值/财务/资金流/公告查询）
- AKShare + 腾讯财经 + 新浪财经（多数据源降级，自动重试）
- httpx（异步调用腾讯云 LKEAP API，三模型并行）
- pandas / numpy（量化计算）
- python-jose + passlib（JWT 认证 + 密码哈希）
- aiofiles（异步文件上传）

**前端**：
- React 18 + Vite（构建工具）
- Recharts（价格走势和评分趋势图表）

**部署**：
- Docker + Docker Compose
- Nginx 反向代理（限流 + 静态缓存 + PDF代理优化）

## 快速开始

### 1. 环境准备

```bash
git clone <repo-url>
cd "real estate stock agent"
```

### 2. 配置环境变量

```bash
cd backend
cp .env.example .env
```

编辑 `backend/.env`：

```env
LKEAP_API_KEY=你的LKEAP_API_Key

DEEPSEEK_MODEL=deepseek-v3.2
GLM_MODEL=glm-5
KIMI_MODEL=kimi-k2.5

JWT_SECRET=修改为随机字符串
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123

# 同花顺iFinD API
IFIND_REFRESH_TOKEN=你的refresh_token
IFIND_ACCESS_TOKEN=你的access_token
```

### 3. Docker 部署（推荐）

```bash
docker compose up -d --build
```

访问 http://localhost，使用管理员账号登录。

### 4. 本地开发运行

**启动后端**：

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**启动前端**：

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000（Vite代理后端API至8000端口）

### 5. 默认账号

首次启动自动创建管理员：`admin` / `admin123`（可通过 .env 修改）。

普通用户由管理员在"用户管理"页面创建，最大支持 100 个账户。
