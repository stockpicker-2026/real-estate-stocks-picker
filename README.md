# 房地产股票 AI 评级 Agent

基于**量化技术分析 + 腾讯混元2.0大模型 + 实时新闻资讯**的中国房地产股票智能评级系统，覆盖 A股、港股、美股三大市场共 55 只房地产相关股票。支持多数据源自动降级、AI联网搜索最新政策，政策资讯对评级的影响权重高达 35%。

## 功能模块

### 1. 用户认证系统
- JWT Token 认证，支持多终端同时登录，token 有效期 72 小时
- 管理员后台创建用户，最大支持 100 个账户
- 角色权限：管理员（全部功能）/ 普通用户（查看功能）

### 2. AI 智能评级
- 量化技术评分 × 50% + AI大模型评分 × 50%
- 每日 09:00 和 16:00 双时段自动刷新，覆盖 55 只房地产股票
- **实时新闻注入**：抓取官方权威媒体（中国政府网、人民网、新华网）及财经媒体最新房地产新闻，经 AI 筛选后注入评级 prompt
- **混元联网搜索**：开启 EnableEnhancement，AI 可实时搜索最新政策信息
- 评分精度精确到小数点后 2 位
- AI 分析状态实时提示：显示 AI+量化混合评级是否成功，以及最近更新时间
- 支持按市场、评级筛选，支持历史日期回溯
- 点击"跟踪股票"、"已评级"统计卡片可弹出股票清单弹窗

### 3. 新闻资讯
- **官方权威数据源**：中国政府网（政策库）、人民网房产频道、新华网房产频道，优先展示权威信息
- **财经媒体补充**：新浪财经房产频道、东方财富房地产板块，作为兜底数据源
- **AI 智能筛选**：通过混元大模型对新闻进行相关性评分，仅保留与房地产高度相关的 Top 5 条
- **关键词兜底排序**：AI 不可用时，基于来源权重 + 政策关键词密度自动排序
- **个股新闻搜索**：东方财富搜索 API，获取每只股票的最新相关新闻
- **缓存机制**：新闻数据缓存 1 小时，避免重复请求
- **前端展示**：可折叠的"房地产政策资讯"板块，精选 5 条，5 分钟自动刷新

### 4. 多数据源降级
- **三层降级架构**：akshare（东方财富）→ 腾讯财经 → 新浪财经
- **智能数据源排序**：连续失败超过 3 次的数据源自动降低优先级
- A股：akshare → 腾讯 → 新浪；港股：akshare → 腾讯；美股：akshare → 腾讯

### 5. 市场点评
- 管理员发布每日行业点评和个股点评
- 支持分类筛选（行业/个股），可关联股票代码
- 支持创建、编辑、删除
- **支持分享到微信朋友圈**（生成公开链接，无需登录即可查看）

### 6. 研究报告
- 管理员上传机构研究报告（PDF/Word/Excel/PPT，最大50MB）
- 用户查看报告列表并下载
- 文件使用 UUID 重命名存储，安全可靠
- **支持分享到微信朋友圈**（含报告摘要和下载入口）

### 7. 微信分享
- 点评和报告均支持一键生成分享链接
- 分享页面包含 Open Graph meta 标签，微信自动抓取标题和描述
- 公开页面无需登录，移动端适配
- 底部引导按钮可跳转至完整系统

## 评级模型

### 综合评分 = 量化技术评分 × 50% + AI大模型评分 × 50%

**量化技术评分**（5个维度加权）：

| 维度 | 权重 | 指标 |
|------|------|------|
| 趋势评分 | 25% | 四级均线排列(MA5/10/20/60)、价格偏离度、MA20斜率、均线黏合度、ADX趋势强度 |
| 动量评分 | 20% | RSI双周期(6+14)、MACD信号+柱状体变化、KDJ随机指标、Williams %R、多周期涨跌幅 |
| 波动率评分 | 15% | 年化波动率、布林带宽度+价格位置、ATR(14)相对波动、波动率收敛/发散趋势 |
| 成交量评分 | 20% | 多级量比(5/10/20日)、OBV能量潮、VWAP偏离度、量价配合度、成交量趋势 |
| 价值评分 | 20% | 区间位置连续评分、筹码集中度、多级支撑压力(10/20/60日)、价格动态区间 |

**AI大模型评分**（腾讯混元2.0 Think + 联网搜索）：
- 政策资讯影响（权重35%）：最新房地产政策、调控文件、行业新闻、市场供需变化，实时联网获取
- 公司基本面（权重30%）：经营质量、财务健康(三道红线)、土储质量、销售回款、管理层能力
- 技术面与资金面（权重25%）：价格趋势、机构资金动向、北向资金、筹码结构分析
- 风险评估（权重10%）：债务违约风险、项目交付风险、系统性行业风险

**评级映射**：

| 综合评分 | 评级 |
|----------|------|
| ≥ 65 | 优选 |
| 50 - 64 | 关注 |
| 35 - 49 | 中性 |
| < 35 | 谨慎 |

> 若 AI 不可用，自动降级为 100% 量化评分，前端会提示当前处于"纯量化分析模式"。

## 技术栈

**后端**：
- FastAPI + Uvicorn（异步 Web 框架）
- SQLAlchemy + aiosqlite（异步 ORM + SQLite）
- APScheduler（定时任务，每日 09:00 + 16:00 双时段自动评级）
- AKShare + 腾讯财经 + 新浪财经（多数据源降级，自动重试）
- httpx（异步调用腾讯混元 API，支持联网搜索）
- pandas / numpy（量化计算）
- python-jose + passlib（JWT 认证 + 密码哈希）
- aiofiles（异步文件上传）

**前端**：
- React 18 + Vite（构建工具）
- Recharts（价格走势和评分趋势图表）

**部署**：
- Docker + Docker Compose
- Nginx 反向代理（限流 + 静态缓存）

## 快速开始

### 1. 环境准备

```bash
git clone <repo-url>
cd "real estate stock agent"
```

### 2. 配置环境变量

```bash
cd backend
cp .env.example .env
```

编辑 `backend/.env`：

```env
HUNYUAN_SECRET_ID=你的SecretId
HUNYUAN_SECRET_KEY=你的SecretKey
HUNYUAN_MODEL=hunyuan-2.0-thinking-20251109

JWT_SECRET=修改为随机字符串
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123
```

### 3. Docker 部署（推荐）

```bash
docker-compose up -d --build
```

访问 http://localhost，使用管理员账号登录。

### 4. 本地开发运行

**启动后端**：

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**启动前端**：

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

### 5. 默认账号

首次启动自动创建管理员：`admin` / `admin123`（可通过 .env 修改）。

普通用户由管理员在"用户管理"页面创建，最大支持 100 个账户。
